# Unit 4 Demo Notes

Use these features for the 5 to 7 minute ScreenPal demo:

1. Show the GitHub repository and folder structure.
2. Open the Flask project in VS Code.
3. Explain requirements.txt and app.py.
4. Run python app.py.
5. Log in as customer1 and submit a request.
6. Log in as employee1 and update document status or delay reason.
7. Log in as manager1 and show dashboard counts and customer requests.
